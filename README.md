# Local-Server-Menu-Dashboard
Easy menu to orient in your URLs. 

### Why I made this?
I have at home small server with several services and I almost getting lost in those URLs where and what I already have on server. Also I wanted one link where I just click and it will redirect me to right page. I don't have to write the url anymore. Just click onto bookmark and then choose right service.

### What's the plan for the future?
Maybe I'll find better solution than this, but for now. I really like this solution. I would like to implement more functions like add/remove button on page, so u don't have to edit it in file. Add little bit customization on page color picker or something like that. Maybe, server stats and things like that. I would like to make from this easy, clear dashboard with smooth cozy vibe.
